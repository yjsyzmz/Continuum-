# Continuum Privacy Policy

Effective date: May 30, 2026

Continuum is a Chrome extension that helps users resume long Claude.ai tasks by saving visible Claude.ai output locally and generating a handoff prompt.

Continuum is not affiliated with, endorsed by, or sponsored by Anthropic or Claude.

## Summary

Continuum is designed to be local-first:

- No Continuum account is required.
- No conversation content is uploaded to a Continuum server.
- No telemetry, analytics, advertising, or tracking SDK is used.
- No data is sold or shared for advertising.
- Captured task data is stored locally in the browser extension's IndexedDB.

## Data Continuum Processes

Continuum may process the following data when you use Claude.ai:

- Visible text content already displayed on Claude.ai pages.
- Local task metadata, such as task status, timestamps, conversation mapping identifiers, and capture state.
- Generated handoff prompt text created locally from the saved visible task content.

This data is used only to provide the extension's core features:

- Save long Claude.ai task output locally.
- Track local task and recording state.
- Generate a resumable handoff prompt.
- Copy the handoff prompt to the clipboard when you choose to copy it.

## Data Continuum Does Not Collect

Continuum does not intentionally collect or transmit:

- Personal identity information, such as name, email address, phone number, age, or government ID.
- Health information.
- Financial or payment information.
- Authentication information, passwords, security questions, or PINs.
- Location data.
- Browsing history across websites.
- Mouse movement, keystroke logging, or general user activity tracking.
- Hidden Claude reasoning, hidden thinking, or content not visible to the user in the page.

If you type personal or sensitive information into Claude.ai and it appears in visible page content, that visible content may be stored locally by Continuum as part of the local task record. Continuum does not upload that content.

## Local Storage and Retention

Continuum stores task records and recordings in the browser extension's local IndexedDB.

The data remains on your device until one of the following happens:

- You delete the extension's local data.
- You remove the extension.
- You clear browser storage associated with the extension.
- A future Continuum feature provides an in-product delete action and you use it.

## Clipboard Permission

Continuum requests the `clipboardWrite` permission so it can copy a generated handoff prompt to your clipboard when you click a copy action.

Continuum does not read your clipboard.

## Claude.ai Host Permission

Continuum requests access to `https://claude.ai/*` so it can run on Claude.ai pages and process visible Claude.ai output for local task recovery.

Continuum does not request access to all websites.

## Remote Code

Continuum does not use remote code. The extension's JavaScript is included in the packaged extension.

## Third Parties

Continuum does not send captured task data to third-party services.

Your use of Claude.ai itself is governed by Anthropic's terms and privacy policy. Continuum only operates as a browser extension on pages you open.

## Security

Continuum keeps task data local to your browser. However, no software can guarantee absolute security. You should avoid using the extension with highly sensitive information if you do not want that information stored locally in your browser profile.

## Changes

This policy may be updated as Continuum changes. Material changes will be reflected in this document and, where required, in the Chrome Web Store listing.

## Contact

For questions or issues, please use the GitHub repository:

https://github.com/yjsyzmz/Continuum-

# Continuum 隐私政策

生效日期：2026 年 5 月 30 日

Continuum 是一个 Chrome 扩展，用于帮助用户在 Claude.ai 长任务中断后继续工作。它会把 Claude.ai 页面上已经显示给用户的可见输出保存在本地，并生成接力 prompt。

Continuum 不是 Anthropic 或 Claude 的官方产品，也未获得其赞助、背书或合作授权。

## 概要

Continuum 采用本地优先设计：

- 不需要 Continuum 账号。
- 不会把对话内容上传到 Continuum 服务器。
- 不使用遥测、分析、广告或追踪 SDK。
- 不出售数据，也不将数据用于广告。
- 捕获的任务数据保存在浏览器扩展本地 IndexedDB 中。

## Continuum 会处理的数据

当你使用 Claude.ai 时，Continuum 可能会处理以下数据：

- Claude.ai 页面上已经显示给你的可见文本内容。
- 本地任务元数据，例如任务状态、时间戳、conversation 映射标识和捕获状态。
- 根据本地保存的可见任务内容生成的 handoff prompt。

这些数据仅用于提供扩展核心功能：

- 本地保存 Claude.ai 长任务输出。
- 跟踪本地任务和录制状态。
- 生成可恢复任务的接力 prompt。
- 在你主动点击复制时，将接力 prompt 写入剪贴板。

## Continuum 不会收集的数据

Continuum 不会有意收集或传输：

- 个人身份信息，例如姓名、邮箱、手机号、年龄或身份证件号码。
- 健康信息。
- 财务或付款信息。
- 登录凭证、密码、安全问题或 PIN。
- 位置信息。
- 跨网站浏览历史。
- 鼠标移动、键盘记录或通用用户活动追踪。
- Claude 的隐藏 reasoning、隐藏 thinking，或页面上未向用户显示的内容。

如果你在 Claude.ai 中输入个人或敏感信息，并且这些内容出现在页面可见文本中，Continuum 可能会把这些可见内容作为本地任务记录的一部分保存在你的浏览器中。Continuum 不会上传这些内容。

## 本地存储与保留

Continuum 将任务记录和录制内容保存在浏览器扩展的本地 IndexedDB 中。

这些数据会保留在你的设备上，直到发生以下情况之一：

- 你删除扩展的本地数据。
- 你卸载扩展。
- 你清除与扩展相关的浏览器存储。
- 后续 Continuum 提供产品内删除功能后，你主动删除相关数据。

## 剪贴板权限

Continuum 请求 `clipboardWrite` 权限，是为了在你点击复制操作时，把生成的接力 prompt 写入剪贴板。

Continuum 不会读取你的剪贴板。

## Claude.ai 主机权限

Continuum 请求访问 `https://claude.ai/*`，是为了在 Claude.ai 页面上运行，并处理 Claude.ai 的可见输出以实现本地任务恢复。

Continuum 不请求访问所有网站。

## 远程代码

Continuum 不使用远程代码。扩展所需 JavaScript 包含在扩展安装包中。

## 第三方

Continuum 不会将捕获的任务数据发送给第三方服务。

你对 Claude.ai 本身的使用受 Anthropic 的服务条款和隐私政策约束。Continuum 只是作为浏览器扩展在你打开的页面上运行。

## 安全

Continuum 将任务数据保存在你的浏览器本地。不过，任何软件都无法保证绝对安全。如果你不希望某些高度敏感信息保存在本地浏览器 profile 中，请避免在相关任务中使用该扩展。

## 变更

如果 Continuum 后续功能发生变化，本政策可能会更新。重大变化会反映在本文档中，并在需要时同步到 Chrome Web Store 商品详情页。
